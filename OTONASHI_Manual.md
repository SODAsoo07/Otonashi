# **OTONASHI (MutauMaker) 풀버전 상세 가이드**

**OTONASHI**는 인간의 발성 원리를 모델링한 성도 시뮬레이션과 전문적인 오디오 편집 도구를 결합한 하이브리드 음성 합성 환경입니다.

## **1\. 인터페이스 구성 및 기본 조작**

### **1.1 기본 레이아웃**

* **상단 헤더**: 스튜디오, 자음 합성, 성도 시뮬레이터 탭 전환.  
* **왼쪽 파일 보관함**: 프로젝트에 사용할 모든 오디오 소스를 관리하는 공간.  
* **중앙 작업 영역**: 현재 선택된 탭의 상세 기능이 표시되는 메인 캔버스.

### **1.2 파일 관리 (보관함)**

* **파일 불러오기 (+)**: 내 컴퓨터에서 오디오 파일(.wav, .mp3 등)을 업로드합니다.  
* **드래그 앤 드롭**: 보관함의 파일을 **스튜디오 파형 영역으로 드래그**하면 즉시 편집 창에 로드됩니다.  
* **파일 이름 변경 (Edit2)**: 파일명을 클릭하거나 편집 아이콘을 눌러 이름을 수정합니다.  
* **파일 삭제 (X)**: 보관함에서 파일을 완전히 제거합니다.

## **2\. 스튜디오 (Studio) 상세 기능**

### **2.1 편집 툴바 (상단)**

* **Undo2 (실행 취소)**: 마지막 편집 작업을 취소하고 이전 상태로 되돌립니다. (최대 20단계)  
* **Scissors (잘라내기)**: 선택 구간을 잘라내어 클립보드에 저장합니다.  
* **Copy (복사)**: 선택 구간을 클립보드에 복사합니다.  
* **Clipboard (붙여넣기)**: 복사한 소리를 선택 지점에 삽입합니다. (기존 소리는 뒤로 밀려납니다.)  
* **Layers (오버레이)**: 복사한 소리를 기존 소리와 **겹쳐서 혼합**합니다. 소리 길이가 변하지 않고 믹스됩니다.  
* **TrendingUp / TrendingDown (페이드 인/아웃)**: 선택 영역에 소리가 서서히 커지거나 작아지는 효과를 적용합니다.  
* **Eraser (침묵)**: 선택 영역을 소리가 없는 무음 상태로 만듭니다.  
* **MoveHorizontal (시간 늘리기)**: 선택 구간의 속도를 50%\~200% 사이로 조절합니다.  
* **Zap (오토메이션)**: 파형 위에 점을 찍어 시간에 따른 볼륨 변화 곡선을 직접 그립니다.

### **2.2 포먼트 및 공명 편집**

스튜디오 하단에 위치한 이 패널은 시뮬레이터 없이도 기존 소리의 주파수 특성을 정밀하게 보정합니다.

* **F1 (제1 포먼트)**: 목소리의 열림 정도를 조절합니다.  
* **F2 (제2 포먼트)**: 목소리의 전/후방 특성을 조절합니다.  
* **F3 (제3 포먼트)**: 목소리의 선명도와 개성을 결정합니다.  
* **공명 (Resonance/Q)**: 각 포먼트 피크의 날카로움을 결정합니다.

## **3\. 자음 합성 (Consonant Mix)**

짧고 거친 자음(파열음, 마찰음) 소리를 생성된 모음과 정밀하게 결합합니다.

* **Vowel (모음) / Consonant (자음)**: 보관함에서 합칠 두 소리를 선택합니다.  
* **자음 타격 오프셋**: 자음이 시작되는 찰나의 순간을 ms(밀리초) 단위로 미세 조정합니다.  
* **개별 볼륨 조절**: 자음과 모음의 출력 비율을 맞춥니다.

## **4\. 성도 시뮬레이터 (Advanced Tract) 상세 가이드**

인간의 구강 구조를 물리적으로 모델링하여 원천 음성을 생성하는 핵심 모듈입니다.

### **4.1 파라미터별 역할 (물리 조작부)**

1. **혀 위치 (Tongue X)**: 앞뒤 위치 결정. 앞으로 갈수록 밝은 소리(ㅣ), 뒤로 갈수록 어두운 소리(ㅜ, ㅗ).  
2. **혀 높이 (Tongue Y)**: 상하 높이 결정. 낮을수록 입을 벌린 개방적인 소리(ㅏ), 높을수록 닫힌 소리.  
3. **입술 열기 (Lips)**: 값이 클수록 고주파 에너지가 강조되어 소리가 밝아집니다.  
4. **입술 길이 (Lip Length)**: 작은 값은 오므린 입술, 큰 값은 ‘이’ 소리를 발음하는 입모양처럼 긴 입술 모양을 시뮬레이션합니다. 길어질수록 공명 주파수가 낮아지는 '원순 모음' 특성을 가집니다.  
5. **목 조임 (Throat)**: 값이 커질수록 압착된 긴장감 있는 소리가 나며, 성도 하단 라인이 굵게 표시됩니다.  
6. **비성 (Nasal)**: 높을수록 콧소리가 섞이며 먹먹한 음색이 됩니다.

### **4.2 파일 가져오기 및 소스 설정 (핵심 기능)**

시뮬레이터 내부의 음원을 신디사이저가 아닌 **외부 파일**로 교체하여 더 풍부한 합성을 할 수 있습니다.

* **음원 소스 (Base Source)**:  
  * **기본 (Synth)**: 톱니파 신디사이저를 사용하여 깨끗한 가상 음성을 만듭니다.  
  * **파일 (File)**: 보관함의 오디오를 '성대'의 역할로 사용합니다. 예를 들어, **녹음된 목소리나 다른 악기 소리**를 불러와 현재 설정된 혀 위치와 입술 모양 필터를 통과시킬 수 있습니다. 이를 통해 기존 녹음본의 발음을 교정하거나 독특한 음성 변조 효과를 낼 수 있습니다.  
* **노이즈 소스 (Noise Source)**:  
  * **기본 (Generated)**: 화이트 노이즈를 사용하여 숨소리를 만듭니다.  
  * **파일 (File)**: 보관함의 특정 파일을 노이즈 재료로 사용합니다. **특수한 질감의 노이즈나 거친 마찰음 샘플**을 소스로 사용하면, 성도 필터를 거치면서 입 모양에 따라 자연스럽게 필터링되는 사실적인 마찰음을 구현할 수 있습니다.  
* **설정 방법**:  
  1. 왼쪽 보관함에 파일을 먼저 업로드합니다.  
  2. 시뮬레이터 설정 패널의 '음원 소스' 또는 '노이즈 소스' 메뉴에서 '보관함 파일'을 선택합니다.  
  3. 아래에 나타나는 드롭다운 목록에서 업로드한 파일을 선택합니다.

### **4.3 키프레임 및 오토메이션 기능**

* **기록 (CircleDot)**: 현재 시점에 모든 조작 값(입 모양, 혀 위치 등)을 **Keyframe**으로 저장합니다.  
* **선형 보간**: 키프레임 사이를 부드럽게 연결하여 조음의 변화를 만듭니다.  
* **타임라인 스크러빙**: 상단 눈금 영역을 드래그하여 특정 시점의 소리를 즉시 확인합니다.  
* **초기화 (Reset)**: 단일 트랙 혹은 전체 오토메이션 기록을 삭제합니다.

## **5\. 저장 및 내보내기**

* **보관함에 저장**: 현재 결과물을 새 파일로 만들어 보관함에 추가합니다.  
* **WAV 저장**: 스튜디오 탭 상단의 **Save** 버튼으로 실제 파일로 다운로드합니다.
